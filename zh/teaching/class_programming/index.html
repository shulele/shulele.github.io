Advance programming

Jan 1, 2019 teaching

“We Shape Our Tools, and Thereafter Our Tools Shape Us”

Objective

Prerequisite for attendee

  • Basic programming knowledge, such as C/C++, Java, R, python, Matlab, etc.

Reference

Textbooks: None

Useful sources:

  • Yang, Daoqi. C++ and object-oriented numeric computing for scientists and engineers. Springer Science & Business Media, 2000.
  • Kernighan, Brian W., and Dennis M. Ritchie. The C programming language. 2006.
  • Victor Shtern, Core C++ A Software Engineering Approach, Prentice Hall (2000) ISBN: 0130857297
  • Running Linux, 4th Edition, Welsh, Kaufman, Dalheimer, and Dawson
  • Chapman, Stephen J. JAVA for Engineers and Scientists. Prentice-Hall, Inc., 2003.
  • Cornell, Gary, and Cay Stephen Horstmann. Core Java. SunSoft press, 1996.
  • Ghai, Ajit. “Facts and Fallacies of Software Engineering.” Software Quality Professional 5.3 (2003): 38.
  • Karniadakis, George Em, George Karniadakis, and Robert M. Kirby II. Parallel scientific computing in C++ and MPI: a seamless approach to parallel algorithms and their implementation. Vol. 2. Cambridge University Press, 2003.
  • Pacheco, Peter. Parallel programming with MPI. Morgan Kaufmann, 1997.

Grading

Two projects and 10 homework.

  • 60% term project (30% per project)
  • 40% homework (4% each)

Total 100%

Outline of the course

  • Introduction
  • The History and Future of Computing
  • Trends in Programming Languages
  • Linux
  • Utilities: vi, emacs, dbx, make, cvs, and IDE’s
  • Brief Introduction to Software Engineering
  • C++
    • Intro
    • Basics
    • Control and Arrays
    • Pointers
    • Structures, Classes, and Objects
    • Operator Overloading
    • Input / Output
    • Templates
    • C++ vs. Java
    • Brief Discussion of Java, C#, and Objective-C
  • Parallel Computing
    • MPI
    • OpenMP
    • Threads
  • Python
  • Conclusions
